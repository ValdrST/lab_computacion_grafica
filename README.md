## Instrucciones
En la raiz de cada carpeta de las practicas existe un archivo llamado makefile
1. Tener instaladas las bibliotecas correspondientes en tu sistema
- make
- g++
- opengl
- glfw
- glew
1. Ejecutar comando 
```
$ make
```
2. Observar resultado
